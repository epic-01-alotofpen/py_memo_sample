# MEMO_APP_PY

## 概要
このプロジェクトは、PythonのTkinterライブラリを使用して作成されたシンプルなメモ帳アプリケーションです。このアプリケーションでは、テキストファイルを新規作成、開く、保存することができ、さらに検索機能を使ってメモ内の特定の文字列をハイライト表示することが可能です。

:::note info
インフォメーション
Tkinterは、PythonでGUI（グラフィカルユーザーインターフェイス）アプリケーションを作成するためのモジュールです。
:::
